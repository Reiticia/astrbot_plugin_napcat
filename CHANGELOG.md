- **问题**：群管理操作返回"权限不足"错误
  **解决**：确认机器人账号在目标群拥有管理员或群主权限。NapCat 的群管 API 需要对应权限才能调用。

- **问题**：发送邮件失败，提示认证错误
  **解决**：检查插件配置中 `email_sender` 和 `email_authorization_code` 是否正确填写。授权码需要在 QQ 邮箱网页版「设置 → 账户 → POP3/SMTP 服务」中生成，不是 QQ 登录密码。

- **问题**：AI 语音发送后群内无反应
  **解决**：确认 NapCat 版本支持 AI 扩展接口。先通过 `/ai_characters` 指令测试角色列表是否能正常返回，若失败则当前 NapCat 版本可能不支持该功能。

- **问题**：LLM 不主动调用工具
  **解决**：确认系统提示词或人格设定中已告知 LLM 本插件的可用工具。可以在 AstrBot 配置中添加类似"你可以使用 send_message 发送消息、set_group_ban 禁言用户等工具"的提示。

- **问题**：指令列表中的斜杠命令无法使用
  **解决**：所有 `/tool_xxx` 和 `/xxx` 指令需要 AstrBot 管理员权限。请确认发送者在 AstrBot 中被配置为管理员。
